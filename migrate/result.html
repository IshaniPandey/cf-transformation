Deploying application to :tgttype.

Perform the following actions:

  1. Go to the conversion path directory cd :tgtdir.
  2. Generate the docker image:docker build -t :applname. .
  3. Tag the docker image for a docker repository that is accessible to the cluster.docker tag :applname. <registry>/<namespace>:applname.
  4. Store the docker image into a docker repository that is accessible to the cluster.docker push <registry>/<namespace>:applname.
  5. Create linkage for any backend services
  6. Deploy the application into the cluster
  7. Generate the service for the application
